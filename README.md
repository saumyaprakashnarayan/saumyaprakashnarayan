### [IntellexDocs](https://github.com/saumyaprakashnarayan/IntellexDocs) · [Live ↗](https://intellex-docs.vercel.app)
**AI-Powered Document Intelligence Platform**

A production-ready RAG platform for natural-language querying over PDF documents. Upload, index, and interrogate documents through a real-time chat interface with citation-backed responses.

- Semantic search pipeline using `RecursiveCharacterTextSplitter`, Gemini Embeddings, and ChromaDB with metadata filtering
- JWT auth, multi-document querying, document summarisation, and persistent chat history via async FastAPI + PostgreSQL
- Full stack containerised with Docker Compose — frontend, backend, PostgreSQL, ChromaDB

`FastAPI` `LangChain` `Gemini` `ChromaDB` `PostgreSQL` `Next.js 15` `TypeScript` `Docker`

### [FasalAI](https://github.com/saumyaprakashnarayan/fasal_ai) · [Live ↗](https://fasal-ai.vercel.app)
**AI-Powered Smart Agriculture Platform**

An agricultural management system that combines ML-based yield prediction, real-time weather data, and multimodal AI analysis to help farmers make data-driven decisions.

- Plant disease detection and crop-stress analysis via TensorFlow/Keras on image inputs and audio (.wav) spectrograms
- Sustainability metrics computed per IPCC, FAO, and ICAR benchmarks (carbon offset, water efficiency, biodiversity index)
- Supabase (PostgreSQL) backend with Row Level Security and Auth; interactive Recharts dashboards for yield trends and soil composition

`Next.js` `TypeScript` `Supabase` `TensorFlow/Keras` `Recharts` `Google Gemini` `OpenWeatherMap`

### [ParkWatch AI](https://github.com/saumyaprakashnarayan/parkwatch_ai) · [Live ↗](https://parkwatch-ai.vercel.app)
**Spatial Analytics for Parking Enforcement**

A spatial analytics engine processing 298,000+ real parking-violation records from Bengaluru to surface illegal hotspots and guide targeted enforcement.

- DBSCAN clustering with Haversine metric (ε = 150 m, MinPts = 8) to identify statistically significant hotspots
- Vectorised pipeline cut runtime from ~10 minutes to **under 25 seconds**
- Custom 5-factor Congestion Impact Score (CIS 0–100) and BPR-based travel-time delay modelling; rendered on an interactive Leaflet.js map

`Python` `Scikit-learn` `DBSCAN` `Pandas` `NumPy` `Leaflet.js` `Streamlit`
